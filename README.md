# BIF 2026 - Landing Page | Ágora Investimentos

Este projeto consiste em uma **Landing Page (LP)** desenvolvida exclusivamente para o **Bradesco Investment Forum 2026**, um dos eventos mais importantes do setor financeiro. O projeto foi concebido e executado dentro da agência **BBDA** para o cliente **Ágora Investimentos**.

---

## 🎯 Objetivo e Conceito

A LP foi desenhada para transmitir a **autoridade, elegância e sobriedade** que o nicho financeiro exige. O foco principal é a conversão para a transmissão online do evento, mantendo uma experiência de usuário (UX) fluida e visualmente limpa.

* **Estética Minimalista:** Uso estratégico de espaços em branco e uma paleta de cores baseada em tons de verde esmeralda e "midnight green".
* **Interatividade Sutil:** Efeitos de *hover* e transições suaves que não poluem a visão nem cansam o usuário, priorizando a seriedade do conteúdo.
* **Hierarquia Visual:** Tipografia clara e bem estruturada para facilitar a leitura de dados complexos e agendas de eventos.

## 🛠️ Diferenciais Técnicos (Hand-Coded)

Diferente da tendência atual de automação excessiva, este projeto foi **desenvolvido 100% manualmente**, sem o uso de ferramentas de IA para geração de código ou frameworks de terceiros.

1.  **CSS3 Artesanal:** Todo o layout foi construído bloco a bloco, garantindo um código limpo, sem redundâncias e de fácil manutenção.
2.  **Responsividade Fluida:** Em vez de depender apenas de breakpoints fixos, utilizei a função `clamp()` do CSS para garantir que a tipografia e os espaçamentos se adaptem de forma orgânica a qualquer tamanho de tela.
3.  **Layout Híbrido:** Implementação estratégica de **Flexbox** e **CSS Grid** para organizar componentes complexos, como a grade de participantes e o formulário de inscrição.
4.  **Performance:** Por ser código nativo e sem bibliotecas externas desnecessárias, a página apresenta um carregamento extremamente rápido.

## 💎 Fidelity & Design-to-Code

Um dos pontos centrais deste desenvolvimento foi a fidelidade rigorosa ao design proposto no **Figma**.
* **Pixel Perfect:** Implementação precisa das medidas, espaçamentos e tokens de design, garantindo que o produto final seja uma réplica exata da visão criativa.
* **Consistência Visual:** Transposição exata de tipografia e paleta de cores, respeitando as guias de estilo da marca Ágora.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Semântico e acessível.
* **CSS3:** Variáveis nativas (Custom Properties), Grid, Flexbox e funções matemáticas (`clamp`, `calc`).
* **JavaScript (Vanilla):** Lógica eficiente para manipulação de DOM e interatividade.
